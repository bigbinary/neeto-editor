# Attachments

`import { Attachments } from "@bigbinary/neeto-editor";`

The Attachments component allows users to upload files by either drag-and-drop or by selecting them through a file input. It utilizes the `Uppy` package for handling the file upload process.

### **Default**

The attachments prop is an array of objects with the following structure:

```js
[
  {
    filename: 'example.jpg',
    url: 'https://example.com/images/example.jpg',
    signedId: 'ASDF1234'
  },
  {
    filename: 'example2.jpg',
    url: 'https://example.com/images/example2.jpg',
    signedId: 'QWER5678'
  },
];
```
