### **Without using the EditorContent component**

#### **Include CSS in your project**

Add the following line to the `<head></head>` tag of your HTML file.

```html
<link
  href="https://cdn.jsdelivr.net/gh/bigbinary/neeto-editor@master/dist/editor-content.min.css"
  rel="stylesheet"
/>
```

> The CDN
> https://cdn.jsdelivr.net/gh/bigbinary/neeto-editor-tiptap@master/dist/editor-content.min.css
> is depreciated and should not be used anymore.

Use the above CDN link to include the styles in your project. Add the class
**neeto-editor-content** to the wrapper of the output content.

#### **Apply syntax highlighting**

Add the following two lines of code to the `<head></head>` tag of your HTML
file.

```html
<link
  href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/10.7.2/styles/github.min.css"
  rel="stylesheet"
/>
<script
  defer
  type="text/javascript"
  src="https://cdn.jsdelivr.net/gh/bigbinary/neeto-editor@master/dist/codeBlockHighlight.js"
></script>
```
