## 1.  Image Wrapper vs. Video Wrapper

In neeto-editor, the `neeto-editor__image-wrapper` and `neeto-editor__video-wrapper` class names serve distinct
purposes:

- **Image Wrapper (`neeto-editor__image-wrapper`):**
The `neeto-editor__image-wrapper` class is assigned to the `div` wrapper of media content directly uploaded from the
editor using the image-upload or video-upload addon. This ensures the distinct styling of the media content within the
editor.

- **Video Wrapper (`neeto-editor__video-wrapper`):**
  The `neeto-editor__video-wrapper` class is utilized for the wrapper of `iframes` that encapsulate embedded videos from
  platforms like YouTube, Loom, etc. This class is applied when a user pastes a video URL directly into the Neeto
  Editor, maintaining a consistent style for embedded videos.
