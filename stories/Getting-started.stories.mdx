# Getting started

<br />
<br />

### **Installation**

<div className="sbdocs-install-code">

```
yarn add @bigbinary/neeto-editor
```

Add the **highlight.js** package as a peer dependency.

```
yarn add highlight.js@11.6.0
```

</div>

### **Usage**

```jsx
import { Editor } from "@bigbinary/neeto-editor";
```

### **Local development**

1. Clone the [repository](https://github.com/bigbinary/neeto-editor-tiptap) and
   switch to the new directory.

2. Run `yarn install`. This will install all the required dependencies.

3. Run `yarn start`. This will watch for local changes as well as spin up
   Storybook.

4. Storybook should now be running and can be visited at
   [http://localhost:6006/](http://localhost:6006/).

### **Building**

The neetoEditor package gets auto-published to npm for every new merge to the
master branch. You can checkout the `publish` workflow in git actions to get a
live update.
