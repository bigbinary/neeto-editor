# Error state

neetoEditor can be configured to provided a custom `error` prop which will be
rendered in the editor. The error prop can be a string, array of strings or an
object with a `message` property.

<Canvas>
  <Editor
    error={{
      message: "Something went wrong in the editor",
    }}
  />
</Canvas>
