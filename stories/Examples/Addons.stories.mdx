Neeto editor enables these options by default.

- `font size`
- `font color`
- `bold`
- `italics`
- `underline`
- `strike through`
- `link`
- `bulleted list`
- `numbered list`

Additional options can be enabled by passing an array of strings to the `addons`
prop.
